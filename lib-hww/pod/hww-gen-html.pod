
=head1 NAME

    hww-gen-html - generate htmls from entry files


=head1 SYNOPSIS

    perl hww.pl gen-html [-i|--update-index] [-m|--missing-only] [--] [<directory> <directory>|<file> <file>]



=head1 DESCRIPTION

if two paths are both directories (or second path doesn't exist),
generate htmls in second directory from entry files in first directory

if two path are both files (or second path doesn't exist),
generate a html to second path from first path.

and if you give B<-i> option, exec B<update-index> command
after this command performed.


=head1 OPTIONS

=over

=item -i, --update-index

if this option was given and if two directories were given,
this performs 'perl hww.pl update-index <destination directory>'.

or if this option was given and if two files were given,
this performs 'perl hww.pl update-index <directory of destination file>'.

=item -I <template file>

this performs 'perl hww.pl update-index <template file> <destination directory>'.

=item -m, --missing-only

generate only missing html files.

=back
