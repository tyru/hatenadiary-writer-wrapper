
=head1 NAME

    hww-update-index - テンプレートファイルからHTMLを生成する


=head1 SYNOPSIS

    perl hww.pl update-index [-m <length> | --max-length <length>] [-i | --update-index] [-I <template file>] [--] [<html directory> | <template file>]


=head1 DESCRIPTION

index.htmlか他のファイル名のHTMLファイルをテンプレートファイルから生成する。
このコマンドはいくつかのパラメータを定義し、
HTML::Templateによってその値を元にテンプレートファイルを生成する。


=head1 THE SAMPLES OF THE TEMPLATE FILE

    </a>

    <br />
    <!-- uncomment the following if you want to show the simple summary. -->
    <!-- <TMPL_VAR NAME="summary"> -->
  </TMPL_LOOP>

=head1 OPTIONS

=over

=item -m <length>, --max-length <length>

要約の文字数分の最大値を決める。
しかし現在このオプションの値はバイト値として扱われる。

=back
